% 	Eine Abschlussarbeit muss nicht zwingend einen Anhang enthalten.

% 	Der Anhang soll dazu genutzt werden, um die eigene Argumentation zu stärken. Er beinhaltet
%	hauptsächlich ergänzende und keine essentiell wichtigen Informationen.


% 	Mögliche Inhalte:
%		- Fragebogen
%		- Transkripte mündlicher Interviews
% 		- Vorlagen für Umfragen und Experteninterviews
%		- Beweis für Forschungsergebnisse in Form von Tabellen und Abbildungen
%		- Ergänzende oder umfassende Tabellen
%		- Weiterführende Abbildungen
%		- Grafiken die den Lesefluss stören
%		- Umfangreiche Texte wie z. B. Definitionen oder fremdsprachige Texte die für die Arbeit
%	 	  wichtig sind und als Ergänzung dienen
%		- Quellcode
%		- Statistiken

% 	Ziele:
%		- Argumente verstärken
%		- Platz für ergänzende / detaillierte Informationen gewähren
%		- Transparenz gewährleisten
%		- Entlastung des Hauptteils
%		- Besseres Textverständnis ermöglichen
%		- Lesefluss verbessern

%	Formatierung:
%		- Muss im Inhhaltsverzeichnis genannt werden
%		- Vor den Inhalten muss ein Blatt mit dem Titel "ANHANG" eingefügt werden
%		- Die Seiten müssen mit römischen Zahlen nummeriert werden. Der Anhang knüpft an die
%		  Nummerierung der vorderen Seiten an.
%		- Bei einem umfangreichen Anhang muss ein separates Inhaltsverzeichnis erstellt werden
%		- Verweise auf den Anhang müsssen im Hauptteil eingefügt werden
